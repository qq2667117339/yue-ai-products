# Contributing to Yue

Thanks for your interest in contributing! This is a small project, but feedback and ideas are welcome.

## How to Contribute

### Bug Reports & Feature Requests

Open an [issue](https://github.com/qq2667117339/yue-ai-products/issues/new) with:

- Clear title and description
- Steps to reproduce (for bugs)
- Expected vs actual behavior

### Code Contributions

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-idea`
3. Make your changes
4. Run `pip install -e .` to verify it installs
5. Push and open a Pull Request

### Style

- Keep it simple. No unnecessary abstractions.
- Follow Python's standard formatting (the project is small, no formatter required yet).
- Comments are welcome where logic isn't obvious.

## Questions?

Open a [Discussion](https://github.com/qq2667117339/yue-ai-products/discussions) (if enabled) or an issue.
